Embedded Weather Station Device

Role: Guided Engineer            Technologies: C/C++, Arduino IDE, OpenWeatherMap, ESP8622 Wemos Board, TFT Touch Display, 3D Printing           Timeline: ~1 month

This is a project I wanted to try out in order to familiarize myself more with embedded systems. The project was a guided github project by Thingplus. The purpose of the device is to have a stable way of taking a quick glance at a weather forcast without having to stop what I'm doing on my computer in order to open a weather app or a website. The process involved behind the construction of the device was done by 3d printing, sodering pins together, downloading source code files, accessing a host API key, installing ports, and uploading the final program sketch using an IDE. In the end, the simple device was able to provide a thurough description of a weather forcast. The user interface details time, date, current tempature celcius, 7-day forcast, and moon phase.
