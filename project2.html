Command Prompt Game / Simulated Processor Execution

Connect4: C to Mips Assembly

Role: Programmer     Technologies: C Programming Language, Geany IDE, Mips Assembly Language, QtSpim Processor Simulator, Linux Ubuntu, Command Prompt     Timeline: ~3 Weeks

This is a fun project I was assigned to go in ECPE 171 Computer Systems and Networks in which I was given the task to write a Connect 4 game (5-in-a-row Varient) in C, and then translate the same game logic over to Mips Assembly. I had a particularly enjoyable time doing this because I this was when I was introduced to a low-level language like Mips. After completing the game in C, I moved on to doing a few function translations from C to Mips, Then once I felt a bit more confident with Mips, I began the translation. I envisioned every move the project as a flow chart, carfully detailing the nessesary commands needed to make acurate transitions. With a lot of patience and focus, I was able to sussessfully translate the game and further gainied experience in writing in assembly. I ran every bit of Assembly code I typed as i went through with the project on the QtSpim Processor simulator untill ultimatly comming up with a final translation.
